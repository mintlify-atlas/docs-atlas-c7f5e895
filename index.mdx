Steamtools is a fast, open-source desktop application written in Rust that lets you manage Steam Lua Manifests directly from a clean GUI. Browse your manifest library, install new manifests by app ID, remove them when needed, and unlock games — all without touching the command line.

## Get started in three steps

<Steps>
  <Step title="Install prerequisites">
    Install the [Visual C++ Runtime (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe) and ensure your GPU drivers include OpenGL support.
  </Step>
  <Step title="Download and launch Steamtools">
    Download the latest release from [GitHub](https://github.com/RealViper8/Steamtools/releases) and run `stcli.exe`.
  </Step>
  <Step title="Set your Steam path">
    On first launch, Steamtools asks for your Steam installation directory. Point it at the folder containing `steam.exe` and click **Validate**.
  </Step>
</Steps>
