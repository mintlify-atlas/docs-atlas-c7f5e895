Steamtools is a fast, open-source desktop application written in Rust that gives you a visual interface for managing Steam Lua Manifests. Instead of manually copying files into Steam's config directory, you load a `.lua` file and Steamtools handles the rest — displaying your manifests as a browsable game grid complete with Steam header images.

## What problems it solves

Steam Lua Manifests live inside `Steam\config\stplug-in\` and must be named by app ID (e.g. `480.lua`). Managing them by hand means navigating the filesystem, tracking which manifests are loaded, and remembering app IDs. Steamtools brings all of that into one window.

## Platform support

Steamtools is a **Windows-only** application. It reads your Steam installation path from the Windows registry automatically, so setup requires nothing more than pointing it at your Steam folder on first launch.
